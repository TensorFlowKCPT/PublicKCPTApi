Документация для api расписания


Получение списка всех групп

{{ data.link }}/groups

Возвращает список групп массивом в формате json

Пример

[
"АТ 21-11-1",
"АТ 21-11-2",
"АТ 21-11-3",
...
]

Получение списка всех преподавателей

{{ data.link }}/teachers

Возвращает список преподавателей словарем где ключ = id преподователя, для удобства id есть и внутри объекта преподователя

Пример


"1": {
"id": 1,
"FirstName": "В.",
"SecondName": "А.",
"Surname": "Ткачук"
}
"2": {
"id": 2,
"FirstName": "В.",
"SecondName": "С.",
"Surname": "Белан"
}
"3": {
"id": 3,
"FirstName": "И.",
"SecondName": "В.",
"Surname": "Русских"
}

Получение списка всех возможных предметов

{{ data.link }}/subjects

Возвращает список предметов массивом в формате json

Пример

[
"Иностранный язык",
"Программирование",
...
]

Получение списка всех кабинетов

{{ data.link }}/classrooms

Возвращает список кабинетов массивом в формате json, включая актовый и спортзал

Пример

[
"101",
"102",
"103",
...
]

Получение списка изменений по дате

{{ data.link }}/changes_day?date={{ data.today }}

Дата должна быть именно в таком формате, (%Y-%m-%d)

Возвращает json словарь изменений для всех групп по дате, где ключ = id изменения, для удобства id есть и внутри объекта изменения

Delete = Номер урока, который это изменение удаляет из расписания (Если оно это делает)

Пример

{
"1": {
"id": 1,
"IsChange": true,
"Date": "2023-09-18",
"Group": "ИСиП 21-11-3",
"Urok": 6,
"Classroom": "891",
"Prepod": "Ткачук В. А.",
"Subject": "Разговоры о важном",
"Comment": "Будет",
"Delete": null
},
"2": {
"id": 2,
"IsChange": true,
"Date": "2023-09-18",
"Group": "ИСиП 21-11-3",
"Urok": 6,
"Classroom": "891",
"Prepod": "Ткачук В. А.",
"Subject": "Иностранный язык",
"Comment": "Будет",
"Delete": null
},
"3": {
"id": 3,
"IsChange": true,
"Date": "2023-09-18",
"Group": "ИСиП 21-11-3",
"Urok": 6,
"Classroom": "891",
"Prepod": "Ткачук В. А.",
"Subject": "Программирование",
"Comment": "Будет",
"Delete": 1
},
...
}

Получение расписания сразу с изменениями на день

{{ data.link }}/class_day?date={{data.today}}&group=ИСиП 21-11-3

Возвращает список уроков на определенную дату и группу

Пример

{
"5": {
"Subject": "Иностранный язык",
"Prepod": "Ткачук В. А.",
"Group": "ИСиП 21-11-3",
"Date": "2023-09-18",
"Number": 5,
"Classroom": "891",
"IsChange": false
},
"6": {
"Date": "2023-09-18",
"Group": "ИСиП 21-11-3",
"Number": 6,
"Classroom": "891",
"Prepod": "Ткачук В. А.",
"Subject": "Программирование",
"Comment": "Будет",
"Delete": null,
"IsChange": true
}
}
